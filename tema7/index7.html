Задачи на создание списков
  1. Создайте список из 20 пятерок
  2. Создайте список целых чисел от -10 до 10 с шагом 2
  3. Создайте список из сумм троек чисел от 0 до 10, используя генератор списка

  4. Заполните массив элементами арифметической прогрессии. Её первый элемент, разность и количество элементов нужно ввести с клавиатуры.

    Формула для получения n-го члена прогрессии: an = a1 + (n-1) * d

    5. Задачи на обработку списков
  5. Заполните список случайными числами в диапазоне 20..100 и подсчитайте отдельно число чётных и нечётных элементов. Использовать цикл.
  6. Дана последовательность действительных чисел а1, а2,..., аn. Заменить все ее члены, большие данного Z, этим числом. Подсчи­тать количество замен.
  7. Дана последовательность чисел, среди которых имеется несколько нулей. Вывести на печать все числа до первого нуля включительно.
  8. Задан линейный массив, состоящий из n элементов. Найти наибольший среди положительных элементов, стоящих на четных метах, и вывести его значение и индекс данного элемента на экран.
  9. В целочисленной последовательности есть нулевые элементы. Создать массив из номеров этих элементов.
  10. Определить индексы элементов массива, значение которых лежит в указанном пределе
  11. В заданный массив целых чисел вставить число 0 после каждого нечетного элемента

  12. Дан одномерный массив из 20 целых чисел (каждое действие выполняется над копией исходного массива):

    * подсчитайте наибольшее число одинаковых идущих подряд в нем чисел;

    * удалить все элементы, кратные последнему элементу массива;

    * в данной последовательности определите число соседств двух чисел разного знака;

    * вывести на печать только те числа, для которых выполняется условие ai<=i.

  13. В массиве действительных чисел все нулевые элементы заменить на среднее арифметическое всех элементов массива.
  14. Дан одномерный массив из 8 элементов. Заменить все элементы массива меньшие 15 их удвоенными значениями. Вывести на экран монитора преобразованный массив.

  15. Дан массив целых чисел. Выполнить с массивом следующие действия:

    * вставить новый элемент перед вторым элементом в существующем массиве

    * удалить указанный элемент, используя индекс из массива

    * удалить первое вхождение указанного элемента из массива

  16. По фигурному катанию N судей выставляют оценки. Наибольшая и наименьшая отбрасываются, а по остальным выставляется средний бал. Напишите программу, вычисляющую среднюю оценку и выдайте номер судей, которые поставили наибольшую и наименьшую оценки.
